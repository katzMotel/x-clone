This Project is a clone of X.com(twitter).  It is written in typescript using React and NextJS and styled with SASS to create a mockup of the current sign-in, 
sign-up, and homepage/feed of x.  There is limited functionality, tweets can be added/deleted.  Tweets can be liked, however the functionality for like count
is not currently coded into the project.  This project can be viewed in vercel or run on your local machine.
